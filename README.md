# Wiki Search Engine

A simple web-based Wikipedia search engine built with HTML, CSS, and JavaScript. It allows users to search for articles on Wikipedia and displays the results in a user-friendly interface.

## Live Demo

[View Live Demo](https://gagankalakota.ccbp.tech)

## Features

- Search for Wikipedia articles by keyword
- Display search results with titles and snippets
- Responsive design using Bootstrap

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap

## How to Use

1. Open `wikipedia search engine.html` in a web browser.
2. Enter a keyword in the search input and press Enter.
3. View the search results below.
